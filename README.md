# CloudComputingProject

Simple social network webapp prepared as a project for Cloud Computing classes.

## Functionalities
- Register new user
- Log user in/out
- Show user's profile
- Edit user's data
- Search for new friends
- Show suggested friends
- Add new friend
- Delete friend
- Show list of friends

## Used technologies
- Backend
  - Python Flask
  - Neo4j database
  - REST
- Frontend
  - JavaScript
  - CSS & Bootstrap
  - HTML
